Status badges

  • pending not yet scheduled — waiting for deps or slot
  • in_progress worker is running in a worktree
  • in_review PR is open; reviewer / iterator may act
  • merged PR merged to main — terminal
  • blocked halted; cascade-blocks dependents

Alert severity

  • ❗ error — needs operator attention (blocked task, dead orchestrator)
  • ⚠ warn — sensitive merge held, urgent monitor finding
  • ⓘ info — slower-burn monitor finding
